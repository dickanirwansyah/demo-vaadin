This zip file contains your personal development license keys for Vaadin Pro products.

To install the licenses, copy them to your home directory.

On Windows, this is typically \Users\<username>. You can install the licenses from the command prompt using:
cp vaadin*license %userprofile%

On Mac, this is typically /Users/<username>. You can install the licenses from the command prompt using:
cp vaadin*license ~

On Linux, this is typically /home/<username>. You can install the licenses from the command prompt using:
cp vaadin*license ~


If you want to hide the development files from view, you can add a '.' as the first character of the filename, e.g. rename 'vaadin.charts.developer.license' to '.vaadin.charts.developer.license'
